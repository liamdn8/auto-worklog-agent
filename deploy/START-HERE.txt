================================================================================
  AUTO WORKLOG AGENT - START HERE
================================================================================

✅ This binary works on ALL Linux distributions!
   (Ubuntu, Debian, Fedora, RHEL, CentOS, Arch, openSUSE, Alpine, etc.)

✅ Statically linked - No dependencies needed!
   (No Python, no libraries, just one 12MB binary file)

================================================================================
  QUICK START (3 steps)
================================================================================

1. Make scripts executable:
   chmod +x quick-setup.sh awagent install.sh

2. Run the setup script:
   ./quick-setup.sh

3. Start the agent:
   ./awagent --verbose

That's it! View your activity at http://localhost:5600

================================================================================
  WHAT YOU NEED
================================================================================

REQUIRED:
- Linux x86_64 (any distribution)
- ActivityWatch server running at http://localhost:5600
- One of these tools (usually pre-installed):
  * xdotool (recommended)
  * xprop (most common)
  * wmctrl
  * gdbus (GNOME)
  * qdbus (KDE)

OPTIONAL:
- Systemd (for running as a service)

================================================================================
  INSTALL WINDOW DETECTION TOOL (if needed)
================================================================================

Check what's already installed:
  which xdotool xprop wmctrl

If nothing found, install xdotool:
  Ubuntu/Debian:  sudo apt-get install xdotool
  Fedora/RHEL:    sudo dnf install xdotool
  Arch Linux:     sudo pacman -S xdotool

Or install xprop (lighter):
  Ubuntu/Debian:  sudo apt-get install x11-utils
  Fedora/RHEL:    sudo dnf install xorg-x11-utils

================================================================================
  START ACTIVITYWATCH SERVER
================================================================================

Using Docker (recommended):
  docker run -d --name activitywatch \
    -p 5600:5600 \
    -v ~/.local/share/activitywatch:/data \
    activitywatch/activitywatch:latest

Verify it's running:
  curl http://localhost:5600/api/0/info

================================================================================
  TROUBLESHOOTING
================================================================================

"No window detection method available"
  → Install xdotool or xprop (see above)

"Connection refused to ActivityWatch"
  → Start ActivityWatch server (see above)

"Permission denied"
  → chmod +x awagent

"No repositories found"
  → Edit ~/.config/awagent/config.json
  → Make sure "roots" points to your home directory
